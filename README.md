# Personal Portfolio Website

A fully responsive personal portfolio website built with HTML5 and CSS3.

## Features

- **Responsive Design**: Works on Mobile, Tablet, and Desktop.
- **No JavaScript**: All interactivity (Mobile Menu, Form Validation) is handled via CSS.
- **Modern Styling**: Uses CSS Variables, Flexbox, and Grid.
- **Components**: Sticky Navbar, Hero Section, Project Cards, Contact Form.

## Pages

1. **Home** (`index.html`): Hero section and featured project.
2. **About** (`about.html`): Bio and Technical Skills.
3. **Projects** (`projects.html`): Gallery of work.
4. **Contact** (`contact.html`): Contact form with validation.

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.

## CSS Concepts Used

- Flexbox & Grid
- Position (Sticky, Absolute)
- CSS Variables
- Media Queries
- Pseudo-classes & Animations
